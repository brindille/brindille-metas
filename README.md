# brindille-metas

Play with meta tags

## Installation

```bash
npm install --save brindille-metas
```

## Usage

```js
var metas = require('brindille-metas');
metas.setTitle('Hello world!');
metas.setDescription('Awesome description');
metas.setShareImage('assets/images/share.png');
```
